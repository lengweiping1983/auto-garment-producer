# 服装术语词典

本文档定义本 Skill 中使用的标准服装工业术语，确保 AI、设计师与生产端使用统一语言。

## 裁片类型（Piece Types）

| 英文角色 | 中文名称 | 说明 |
|---|---|---|
| `front_hero` | 前片主片 | 正面最大、最显眼的裁片，通常承载卖点图案 |
| `back_body` | 后片身片 | 背面主体裁片，应安静不抢戏 |
| `secondary_body` | 次主体片 | 较大的辅助身片，如前片副片、后片拼接片 |
| `sleeve_pair` | 袖对 | 左右对称的袖片，应协调或镜像处理 |
| `sleeve_or_side_panel` | 袖片/侧片 | 较大的偏心裁片，可能是袖片或侧缝片 |
| `side_or_long_panel` | 侧片/长条片 | 长条形裁片，起修身或框定作用 |
| `trim_strip` | 饰边条 | 窄长条裁片，用于装饰边缘 |
| `collar_or_upper_trim` | 领片/上饰边 | 位于上方的较小裁片 |
| `hem_or_lower_trim` | 下摆/下饰边 | 位于下方的较小裁片 |
| `small_detail` | 小细节 | 面积很小的裁片，如贴袋、耳仔 |
| `matched_panel` | 配对片 | 形状几乎相同的重复裁片 |

## 部位区域（Zones）

| 英文 | 中文 | 说明 |
|---|---|---|
| `body` | 身片区 | 构成服装主体的较大裁片 |
| `secondary` | 副片区 | 次于主体的可见裁片，如袖片、侧片 |
| `trim` | 饰边区 | 窄条、领、袖口、下摆等边缘处理 |
| `detail` | 细节区 | 面积小、不主导视觉的裁片 |

## 面料角色（Texture Roles）

| 英文 | 中文 | 用途说明 |
|---|---|---|
| `main` | 主面料 | 低噪可穿底纹，覆盖 55–75% 可见面积 |
| `secondary` | 辅面料 | 协调的大副裁片纹理 |
| `accent` | 点缀面料 | 小面板或细节纹理 |
| `dark` | 深色面料 | 袖口、窄条、打底片 |
| `light` | 浅色面料 | 提亮区域 |
| `solid` | 纯色 | 极窄条或需要纯色的区域 |
| `hero_motif` | 卖点图案 | 1–2 个关键裁片上的定位图案 |

## 填充参数术语

| 英文 | 中文 | 说明 |
|---|---|---|
| `fill_type` | 填充类型 | `texture`（面料）、`solid`（纯色）、`motif`（图案） |
| `scale` | 缩放 | 纹理/图案相对于原始尺寸的比例 |
| `rotation` | 旋转 | 顺时针旋转角度（度） |
| `offset_x` / `offset_y` | 水平/垂直偏移 | 纹理平铺的起始偏移 |
| `mirror_x` / `mirror_y` | 水平/垂直镜像 | 是否对纹理进行镜像翻转 |
| `opacity` | 不透明度 | 0–1，1 为完全不透明 |
| `anchor` | 锚点 | 图案定位的参考点（居中、顶部、底部等） |
| `base` | 底层 | 裁片的基础填充层 |
| `overlay` | 叠加层 | 覆盖在底层之上的图案或纹理 |
| `trim` | 饰边层 | 可选的饰边处理层 |

## 对称性与分组

| 英文 | 中文 | 说明 |
|---|---|---|
| `symmetry_group` | 对称组 | 左右镜像的成对裁片（如左右袖） |
| `same_shape_group` | 同形组 | 形状几乎相同的重复裁片（如前后贴袋） |
| `direction_degrees` | 方向角度 | 裁片在服装上的主要方向 |

## 计划字段术语

| 英文 | 中文 | 说明 |
|---|---|---|
| `tileable_score` | 可平铺分数 | 0–1，越高表示边缘越适合无缝平铺 |
| `variation_score` | 变化度分数 | 0–1，越高表示纹理视觉变化越丰富 |
| `confidence` | 置信度 | 部位推断或几何分析的可信程度 |
| `approved` | 可用 | 是否允许当前流程使用 |
| `locked` | 已锁定 | 用户确认后禁止修改 |
