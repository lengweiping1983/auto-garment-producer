# 数据契约

在 AI 风格分析、纹理生成、质检与 OpenCV 渲染之间，使用 JSON 作为数据交换格式。

> 为实现中英双语兼容，以下 JSON 示例中的键名优先使用英文（保持代码兼容性），但在实际输出中，所有 `"reason"`、`"notes"`、`"message"` 等描述性字段均使用中文。部分关键对象在代码层同时支持英文键和中文别名映射。

## 风格档案（style_profile.json）

```json
{
  "style_id": "forest_watercolor_001",
  "art_style": "柔和水彩童话森林",
  "palette": {
    "primary": ["#6f9a4d", "#9fcf70", "#dfe8a6"],
    "accent": ["#8fcbd1", "#f0c45c"],
    "dark": ["#3f5f35", "#6b4f32"]
  },
  "motifs": ["碎花", "叶片", "苔藓", "溪流波纹", "蘑菇"],
  "avoid": ["动物", "人物", "人脸", "文字", "商标", "中心构图"],
  "texture_density": "中等",
  "contrast": "低"
}
```

## 商业设计简报（commercial_design_brief.json）

在资产生成之前创建。记录成衣意图，而非渲染器设置。

```json
{
  "brief_id": "rainbow_meadow_commercial_v1",
  "aesthetic_direction": "商业畅销款打样",
  "garment_type": "儿童外套套装",
  "target_customer": "购买日常 cheerful 服装的儿童与家长",
  "season": "春季",
  "price_tier_signal": "中端精致",
  "hero_selling_point": "柔和彩虹草地氛围",
  "fabric": {
    "has_nap": false,
    "nap_direction": "",
    "notes": "无绒毛面料，旋转和镜像不受限制"
  },
  "wearability_notes": [
    "大身裁片保持低对比度",
    "将彩虹作为受控点缀，而非满版噪声"
  ],
  "avoid": ["直接叙事裁剪", "人脸", "文字", "商标", "过度堆砌的袖口"]
}
```

`fabric` 字段说明：
- `has_nap`: 面料是否有绒毛/方向性（灯芯绒、丝绒、植绒等）。`true` 时所有裁片必须同向旋转，禁止 `mirror_y`。
- `nap_direction`: 绒毛方向（通常与 `grain_direction` 一致）。
- `notes`: 面料工艺备注。

## 图案提示词（motif_prompts.json）

图案是可选的、经过工程化定位的点缀。必须在渲染前获批。

```json
{
  "style_id": "forest_watercolor_001",
  "generation_owner": "external_ai_image_model",
  "motifs": [
    {
      "motif_id": "hero_rainbow_meadow",
      "purpose": "前片卖点定位",
      "prompt": "透明 PNG 图案，简化水彩彩虹与碎花，无动物，无人物，无文字，无商标",
      "negative_prompt": "动物，人脸，文字，商标，水印，复杂背景"
    }
  ]
}
```

## 资产生成清单（asset_generation_manifest.json）

可选的交接文件，用于图像生成步骤。Codex 可创建此文件，但不得伪造生成的图像。

```json
{
  "manifest_id": "forest_asset_generation_v1",
  "status": "waiting_for_ai_generated_assets",
  "required_assets": [
    {
      "asset_id": "main",
      "asset_type": "texture",
      "output_requirement": "无缝可平铺正方形 PNG，至少 1024×1024",
      "prompt_ref": "texture_prompts.json#main"
    },
    {
      "asset_id": "hero_motif",
      "asset_type": "motif",
      "output_requirement": "透明 PNG 定位图案，至少 1024px 宽",
      "prompt_ref": "motif_prompts.json#hero_motif"
    }
  ],
  "notes": [
    "生成的资产必须保存到磁盘，并在面料组合.json 中列出后才能渲染。"
  ]
}
```

## 面料组合（texture_set.json）

渲染器读取此文件。

```json
{
  "texture_set_id": "forest_texture_set_v1",
  "locked": false,
  "textures": [
    {
      "texture_id": "main",
      "path": "/path/to/main_texture.png",
      "role": "main",
      "approved": true,
      "prompt": "无缝可平铺水彩……",
      "model": "external",
      "seed": "12345",
      "qc": {
        "tileable_score": 0.91,
        "approved": true
      }
    }
  ],
  "motifs": [
    {
      "motif_id": "hero_rainbow_meadow",
      "path": "/path/to/hero_rainbow_meadow.png",
      "role": "hero",
      "approved": true,
      "prompt": "透明 PNG 图案……",
      "model": "external",
      "seed": "12345",
      "qc": {
        "approved": true
      }
    }
  ],
  "solids": [
    { "solid_id": "forest_green", "color": "#6f9a4d", "approved": true }
  ]
}
```

必需角色：

- `main`（主面料）
- `secondary`（辅面料）
- `accent`（点缀面料）

可选角色：

- `dark`（深色面料）
- `light`（浅色面料）
- `solid`（纯色）

## 部位映射（garment_map.json）

当没有精确的成衣元数据时，从 `裁片清单.json` 推断生成。

```json
{
  "map_id": "garment_map_v1",
  "method": "geometry_inference",
  "confidence": 0.72,
  "pieces": [
    {
      "piece_id": "piece_001",
      "garment_role": "front_hero",
      "zone": "body",
      "symmetry_group": "",
      "texture_direction": "transverse",
      "texture_direction_hint": "transverse",
      "grain_direction": "vertical",
      "confidence": 0.78,
      "reason": "最大居中裁片"
    }
  ]
}
```

字段说明：
- `texture_direction`: 裁片纹理方向（`transverse` 横向 / `longitudinal` 纵向）。**由 AI 审美子 Agent 决定**，程序未提供时可能为空字符串。
- `texture_direction_hint`: 程序兜底的方向建议，仅供 AI 参考，可覆盖。
- `grain_direction`: 经向方向（`vertical` 人体上下 / `horizontal` 水平 / `bias_45` 斜裁）。基于服装设计学知识推断，body 区通常为 vertical，饰边沿长边方向。
