# 数据契约

在 AI 风格分析、纹理生成、质检与 OpenCV 渲染之间，使用 JSON 作为数据交换格式。

> 为实现中英双语兼容，以下 JSON 示例中的键名优先使用英文（保持代码兼容性），但在实际输出中，所有 `"reason"`、`"notes"`、`"message"` 等描述性字段均使用中文。部分关键对象在代码层同时支持英文键和中文别名映射。

## 风格档案（style_profile.json）

```json
{
  "style_id": "forest_watercolor_001",
  "art_style": "柔和水彩童话森林",
  "palette": {
    "primary": ["#6f9a4d", "#9fcf70", "#dfe8a6"],
    "accent": ["#8fcbd1", "#f0c45c"],
    "dark": ["#3f5f35", "#6b4f32"]
  },
  "motifs": ["碎花", "叶片", "苔藓", "溪流波纹", "蘑菇"],
  "avoid": ["动物", "人物", "人脸", "文字", "商标", "中心构图"],
  "texture_density": "中等",
  "contrast": "低"
}
```

## 主题落地策略（theme_to_piece_strategy）

视觉分析必须给出主题如何落到裁片。目标是“主题可识别但成衣可穿”，不是把主题图直接满版铺到纸样上。

```json
{
  "theme_to_piece_strategy": {
    "base_atmosphere": "大身低噪底纹只继承主题色彩、笔触和氛围",
    "hero_motif": "唯一主卖点元素，建议放在前片或指定 hero 裁片",
    "accent_details": "小花、叶片、蘑菇等只用于小面积点缀",
    "quiet_zones": "袖片、后片、领口、窄条保持安静",
    "do_not_use_as_full_body_texture": ["蘑菇", "动物", "角色", "完整花丛", "完整场景"]
  }
}
```

## 商业设计简报（commercial_design_brief.json）

在资产生成之前创建。记录成衣意图，而非渲染器设置。

```json
{
  "brief_id": "rainbow_meadow_commercial_v1",
  "aesthetic_direction": "商业畅销款打样",
  "garment_type": "儿童外套套装",
  "target_customer": "购买日常 cheerful 服装的儿童与家长",
  "season": "春季",
  "price_tier_signal": "中端精致",
  "hero_selling_point": "柔和彩虹草地氛围",
  "fabric": {
    "has_nap": false,
    "nap_direction": "",
    "notes": "无绒毛面料，旋转和镜像不受限制"
  },
  "wearability_notes": [
    "大身裁片保持低对比度",
    "将彩虹作为受控点缀，而非满版噪声"
  ],
  "avoid": ["直接叙事裁剪", "人脸", "文字", "商标", "过度堆砌的袖口"]
}
```

`fabric` 字段说明：
- `has_nap`: 面料是否有绒毛/方向性（灯芯绒、丝绒、植绒等）。`true` 时所有裁片必须同向旋转，禁止 `mirror_y`。
- `nap_direction`: 绒毛方向（通常与 `grain_direction` 一致）。
- `notes`: 面料工艺备注。

## 纹理提示词（texture_prompts.json）

```json
{
  "style_id": "forest_watercolor_001",
  "generation_owner": "external_ai_image_model",
  "prompts": [
    {
      "texture_id": "main",
      "purpose": "大身裁片",
      "prompt": "无缝可平铺水彩童话森林草地图案，碎花，苔藓绿，奶油黄，柔和手绘，无动物，无人物，无文字，无商标，无中心构图",
      "negative_prompt": "动物，人物，人脸，文字，商标，水印，中心场景，边框"
    }
  ]
}
```

## 图案提示词（motif_prompts.json）

图案是可选的、经过工程化定位的点缀。必须在渲染前获批。

```json
{
  "style_id": "forest_watercolor_001",
  "generation_owner": "external_ai_image_model",
  "motifs": [
    {
      "motif_id": "hero_rainbow_meadow",
      "purpose": "前片卖点定位",
      "prompt": "透明 PNG 图案，简化水彩彩虹与碎花，无动物，无人物，无文字，无商标",
      "negative_prompt": "动物，人脸，文字，商标，水印，复杂背景"
    }
  ]
}
```

## 资产生成清单（asset_generation_manifest.json）

可选的交接文件，用于图像生成步骤。Codex 可创建此文件，但不得伪造生成的图像。

```json
{
  "manifest_id": "forest_asset_generation_v1",
  "status": "waiting_for_ai_generated_assets",
  "required_assets": [
    {
      "asset_id": "main",
      "asset_type": "texture",
      "output_requirement": "无缝可平铺正方形 PNG，至少 1024×1024",
      "prompt_ref": "texture_prompts.json#main"
    },
    {
      "asset_id": "hero_motif",
      "asset_type": "motif",
      "output_requirement": "透明 PNG 定位图案，至少 1024px 宽",
      "prompt_ref": "motif_prompts.json#hero_motif"
    }
  ],
  "notes": [
    "生成的资产必须保存到磁盘，并在面料组合.json 中列出后才能渲染。"
  ]
}
```

## 面料组合（texture_set.json）

渲染器读取此文件。

```json
{
  "texture_set_id": "forest_texture_set_v1",
  "locked": false,
  "textures": [
    {
      "texture_id": "main",
      "path": "/path/to/main_texture.png",
      "role": "main",
      "approved": true,
      "prompt": "无缝可平铺水彩……",
      "model": "external",
      "seed": "12345",
      "qc": {
        "tileable_score": 0.91,
        "approved": true
      }
    }
  ],
  "motifs": [
    {
      "motif_id": "hero_rainbow_meadow",
      "path": "/path/to/hero_rainbow_meadow.png",
      "role": "hero",
      "approved": true,
      "prompt": "透明 PNG 图案……",
      "model": "external",
      "seed": "12345",
      "qc": {
        "approved": true
      }
    }
  ],
  "solids": [
    { "solid_id": "forest_green", "color": "#6f9a4d", "approved": true }
  ]
}
```

必需角色：

- `main`（主面料）
- `secondary`（辅面料）
- `accent`（点缀面料）

可选角色：

- `dark`（深色面料）
- `light`（浅色面料）
- `solid`（纯色）

## 部位映射（garment_map.json）

当没有精确的成衣元数据时，从 `裁片清单.json` 推断生成。

```json
{
  "map_id": "garment_map_v1",
  "method": "geometry_inference",
  "confidence": 0.72,
  "pieces": [
    {
      "piece_id": "piece_001",
      "garment_role": "front_hero",
      "zone": "body",
      "symmetry_group": "",
      "texture_direction": "transverse",
      "texture_direction_hint": "transverse",
      "grain_direction": "vertical",
      "confidence": 0.78,
      "reason": "最大居中裁片"
    }
  ]
}
```

字段说明：
- `texture_direction`: 裁片纹理方向（`transverse` 横向 / `longitudinal` 纵向）。**由 AI 审美子 Agent 决定**，程序未提供时可能为空字符串。
- `texture_direction_hint`: 程序兜底的方向建议，仅供 AI 参考，可覆盖。
- `grain_direction`: 经向方向（`vertical` 人体上下 / `horizontal` 水平 / `bias_45` 斜裁）。基于服装设计学知识推断，body 区通常为 vertical，饰边沿长边方向。

## 多方案生产规划（ai_multi_production_plan.json）

`--dual-source --multi-scheme` 使用该格式。双源都成功时，A/B 两套 3x3 看板会先合并为 18 图资产池，资产 ID 带 `_a` / `_b` 后缀；AI 必须基于完整资产池输出多套独立设计方案，而不是只返回 A/B 两个来源结果。

```json
{
  "schemes": [
    {
      "scheme_id": "scheme_01",
      "design_positioning": "量产安全款",
      "strategy_note": "低噪大身配单一卖点图案",
      "theme_landing_summary": {
        "hero_piece": "piece_001",
        "base_atmosphere_pieces": ["piece_002"],
        "quiet_pieces": ["piece_003"],
        "accent_pieces": ["piece_004"],
        "reason": "主题主体只落在一个 hero 裁片，大身只保留氛围和色板"
      },
      "asset_mix_summary": {
        "body_base_assets": ["main_a"],
        "hero_assets": ["hero_motif_1_b"],
        "trim_assets": ["quiet_solid_a"],
        "source_mix_policy": "single_source_body_with_small_accent",
        "reason": "A 源底纹稳定，B 源主图案更清晰"
      },
      "diversity_tags": ["quiet_body", "bold_hero"],
      "piece_fill_plan": {
        "pieces": [
          {
            "piece_id": "piece_001",
            "base": {
              "fill_type": "texture",
              "texture_id": "main_a",
              "scale": 1.0,
              "rotation": 0,
              "offset_x": 0,
              "offset_y": 0
            },
            "overlay": {
              "fill_type": "motif",
              "motif_id": "hero_motif_1_b",
              "anchor": "center",
              "scale": 0.72,
              "opacity": 0.92
            },
            "trim": null,
            "texture_direction": "transverse",
            "reason": "前片保持清晰卖点，底纹不抢主图案"
          }
        ]
      }
    }
  ],
  "portfolio_notes": "覆盖量产安全、强卖点、深色高级、轻量呼吸等方向",
  "asset_coverage": {
    "used_assets": ["main_a", "hero_motif_1_b", "quiet_solid_a"],
    "unused_assets": [
      {"asset_id": "trim_motif_b", "reason": "噪声较高，不适合量产款"}
    ],
    "coverage_strategy": "保留高质资产，多套方案分别使用不同主次关系"
  },
  "risk_notes": []
}
```

字段说明：
- `schemes`: 必填数组，每项是一套独立方案。
- `scheme_id`: 必填，建议 `scheme_01`、`scheme_02`。
- `design_positioning`: 必填，如“量产安全款”“强卖点款”“深色高级款”。
- `strategy_note`、`theme_landing_summary`、`asset_mix_summary`、`diversity_tags`: 必填，用于说明方案差异、主题落地方式和资产选择理由。
- `piece_fill_plan`: 必填，结构与单方案填充计划一致。
- `garment_map`: 可选。模板模式下会复用固定模板映射；即使 AI 输出也会被模板映射覆盖。
