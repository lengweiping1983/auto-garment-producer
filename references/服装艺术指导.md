# 服装艺术指导

当输出需要看起来像商业成衣，而不仅是技术纹理填充时，参考本文档。

## 商业畅销款默认方向

设计一款能卖得出去的产品：第一眼可读、反复穿着依然舒适、易于生产、视觉上不疲惫。

核心比例：

- 1 个卖点概念，放在 1–2 个关键裁片上。
- 主可穿底纹覆盖 55–75% 的可见面积。
- 辅面料覆盖 15–30%。
- 点缀/图案处理覆盖 5–15%。
- 饰边、袖口、窄条和边缘裁片使用安静的纯色或低噪深色/浅色纹理。

## 设计师决策顺序

1. 确定服装类型与目标客户。
2. 将主题提炼为一个可卖的概念，而非 literal 的故事裁剪。
3. 选择 hero 区域：通常是最大居中身片或一个强势可见面板。
4. 为大裁片选择安静的底纹。
5. 克制地使用图案定位。
6. 用袖口、窄条、下摆和侧缝控制节奏与收尾。
7. 从远处和近处分别检查整体预览。

## 裁片分配启发式规则

- `front_hero` / 最大居中裁片：底纹 + 可选图案叠加。
- `back_body`（后片）：安静的相关底纹或辅面料；避免与前片竞争。
- `sleeve_pair`（袖对）：与身片同系列，镜像或错位，使一对袖子看起来有意设计。
- `side_panel`（侧片）：辅面料或深色纹理，起到修身与框定作用。
- `trim_strip`（饰边条）、`cuff`（袖口）、`hem`（下摆）、`collar`（领片）：纯色、深色、浅色或极安静的点缀。
- `small_detail`（小细节）：低密度点缀或纯色，绝不使用复杂插画。

## 失败案例

以下情况应拒绝或修改方案：

- 所有裁片以相同密度和相同尺度填充。
- 将叙事插画切割到裁片中。
- 在纹理图块中放入动物、人脸、文字或商标。
- 将卖点图案放在不可控的接缝连接处。
- 在窄条或袖口上使用高对比度。
- 让所有裁片同样鲜艳。
- 忽略成对裁片的左右对称性。

## 时尚 QC 检查清单

输出应满足以下全部条件：

- 是否有明确的单一卖点？
- 大裁片是否可穿且低噪？
- 饰边是否比身片更安静？
- 成对裁片是否感觉协调？
- 色板是否保持受控？
- 缝纫后是否仍然可读？

## 成衣视角强制自检

在写 `piece_fill_plan` 之前，必须把所有裁片想象成已经缝合、挂在衣架上、距离 3 米观看的一件完整衣服，并回答以下四问。四问必须进入 `pre_design_self_check` 或 `strategy_note`。

Q1【整体印象】这件衣服第一眼让人记住的是什么？用不超过 12 个字概括。答案必须指向具体视觉锚，例如“胸口白色山猫剪影”，不能只写“主题氛围”或“配色和谐”。如果想不出，说明 hero 不清晰，必须重做。

Q2【三米测试】距离 3 米看，哪些纹理会糊成一团灰或花成一片？大身 base 在 3 米外应塌缩成一个受控色相，而不是杂讯。凡是远看只剩“花、乱、糊”的方案，必须 `approved=false`。

Q3【挂架对比】把这件衣服和同价位 20 件 T 恤或上衣挂在一起，它的上架理由是什么？必须能用一句话说明消费者能读懂的视觉卖点，不能只写“主题独特”。

Q4【日常穿着舒适度】穿这件衣服去开会、吃饭、接孩子，会不会用力过猛？大身基底必须通过日常场景测试。大身满版高对比插画或多源拼贴必须降级为低噪 base。

## Hero Motif 几何契约

Hero 只允许出现在 `front_body` 或 `front_hero`。所有数值相对该裁片 bounding box，而不是整张画布像素。

| 维度 | 下限 | 上限 | 说明 |
| --- | --- | --- | --- |
| 宽度占比 | 18% | 42% | 小于 18% 远看消失，大于 42% 像整张贴纸 |
| 高度占比 | 18% | 38% | 不得压迫领口、肩线或胸围线 |
| 顶端距裁片上沿 | 12% | - | 避开领口和肩缝 |
| 底端距裁片下沿 | 22% | - | 避开下摆和缝合误差 |
| 与任意裁片边最近距离 | 8% | - | 避开袖窿、侧缝、肩缝 |
| 旋转 | -8° | +8° | 超出范围会显得歪斜 |
| motif 与底色明度差 ΔL* | 25 | - | 否则远看糊成一团 |

禁止操作：

- 半透明整张主题图当 overlay。
- motif 横跨缝合线、领口、袖窿、肩缝或前片中线。
- 同一件衣服出现 2 个以上 hero motif。
- 使用完整场景叙事作为 hero，例如草地、房子、动物和树同时出现。只能提取 1 个角色或 1 个简化图标。

违反任一条时，必须改为 fallback safe plan，或重新选择 motif 和位置。不能只靠微调 scale/offset 勉强通过。

## 双源资产硬规则

当资产 ID 带 `_a` / `_b` 后缀时，代表双源资产池。双源不等于必须混搭。

1. 必须先二选一作为 `asset_shortlist.primary_source`。判断顺序为主题贴合度、内部 palette 一致性、笔触一致性、底色一致性。
2. `primary_source` 承担至少 80% 的可见视觉面积。所有大身 base、前后身主体、袖片主底纹必须来自 primary。
3. 另一源只能贡献不超过 2 个小面积资产，且只能用于 `trim`、`cuff`、`collar`、`small_detail` 或低饱和 accent。
4. 另一源若底色、饱和度、笔触或媒介明显不同，必须完全放弃混用。
5. 不能为了“用满 18 张资产”而拼贴。舍弃资产是专业设计决策，必须在 `asset_shortlist.rejected_assets` 写明原因。

## 主题元素 S/A/B/C 分级表

视觉提取阶段必须为 `dominant_objects[]` 标注 `grade`。

S 级：绝不能作为大身满版纹理，只能被拒绝、简化为剪影，或作为很小的定位 motif。包括完整动物、人脸/人像、文字、商标、完整建筑、完整场景、复杂叙事插画。

A 级：只允许 1 个 hero 裁片使用，且必须遵守 Hero Motif 几何契约。包括简化动物剪影、单朵大花、几何图标、单个清晰角色符号。

B 级：可作小面积 accent，最多 3 个裁片。包括小花、小叶、抽象笔触点缀、小型几何元素。

C 级：可作 base 满版，安全。包括主题色彩晕染、无具象形状的抽象纹理、水彩底、低对比噪点底、对比不超过 5% 的小循环几何。

`theme_to_piece_strategy.do_not_use_as_full_body_texture` 必须包含所有 S 级元素和不适合大身的 A 级元素。`generated_prompts.main` 与 `generated_prompts.secondary` 只能继承色彩、笔触、氛围，不得直接包含 S/A 级具象主体。

视觉提取还必须为可用作 hero 的 S/A 级主体给出 `garment_placement_hint`。参考图中的 `geometry` 只描述主体在原图里的尺寸和位置，不能直接当作上身比例。推荐默认值：胸口定位、宽度占目标裁片 0.28–0.34、高度占目标裁片 0.22–0.32；禁止 full bleed、跨肩缝、跨袖窿、跨领口或完整场景满版。

## Wearable Zoning 结构化规则

`references/styles/*-wearable-zoning.jpg` 属于历史效果参考资产，不得进入自动生产规划 prompt 或 Kimi 图片输入。自动流程必须优先使用模板 `garment_map`、`piece_overview` 和下方文字分区规则作为成衣视角锚点。

- `HERO ZONE`：唯一允许放置 hero motif 的区域。
- `QUIET ZONE`：必须使用低噪 base 或纯色，不能放具象主体。
- `DANGER SEAM`：禁止高对比图案、文字、动物、人脸或半透明 overlay。
- `TRIM ZONE`：只允许纯色、低噪深浅色或极小尺度 accent。
- `SLEEVE PAIR`：左右袖必须同源、同密度、同笔触，可镜像或轻微错位，但不能像两套不同设计。

如果模板 `garment_map`、纸样总览和 AI 的部位识别冲突，必须在 `garment_map_uncertainties` 中说明，并采用更保守的安静填充。

## 内置模板文字分区

这些文字分区是自动流程的 zoning 依据。不得依赖 wearable-zoning 图片；AI 必须按以下区域约束执行。

### DDS26126XCJ01L（T 恤/衬衫）

- `HERO ZONE`：前片胸口居中，约为前片宽 28%–34%、高 22%–32%，不得接触领口、肩缝、袖窿或下摆。
- `QUIET ZONE`：前后身大面积区域与袖片主体，必须使用同源低噪 base。
- `DANGER SEAM`：领口、肩缝、袖窿、侧缝、前片边线，禁止高对比 motif、文字、动物、人脸和半透明贴片。
- `TRIM ZONE`：领条、下摆、袖口、窄条，只允许纯色、低噪深浅色或极小尺度 accent。
- `SLEEVE PAIR`：左右袖必须来自同一 source、同一 base 家族、同一密度和笔触，可镜像或轻微错位。

### BFSK26308XCJ01L（防晒服/外套）

- `HERO ZONE`：左胸或右胸小面积定位，宽度不超过对应前身片 34%，高度不超过 30%，避免门襟和拉链中心线。
- `QUIET ZONE`：前后身大面、长袖主体和背部必须安静，不允许完整场景或满版高密度图案。
- `DANGER SEAM`：领口、肩缝、袖窿、前中门襟/拉链、袖下缝和侧缝，禁止高对比图案跨越。
- `TRIM ZONE`：门襟、袖口、下摆、帽/领边，只允许最低饱和度 accent 或 solid。
- `SLEEVE PAIR`：两只长袖必须同源同密度，不能一只使用强图案、一只使用安静底纹。
