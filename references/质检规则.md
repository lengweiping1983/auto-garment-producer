# 质检规则

纹理质检是可靠性闸门。OpenCV 渲染只能使用已批准的纹理资产。

## 自动检查

基础版 `质检纹理.py` 检查以下内容：

- 文件存在且可打开
- 图像尺寸足够大
- 纹理具有足够的视觉变化
- 对边足够相似以保证平铺
- 存在 approved 标志

## 必需的人工或 AI 检查

还需检查：

- 无动物
- 无人物/人脸
- 无文字/商标/水印
- 无强烈中心构图
- 无可见边框
- 色板与主题匹配
- 纹理密度与分配的裁片兼容
- 大身裁片比点缀裁片更低噪
- 窄条/袖口不过度堆砌
- 卖点图案出现在不超过 1–2 个关键裁片上
- 成对裁片协调或有意镜像

## 审批闸门

仅允许具有以下状态的面料通过：

```json
{
  "approved": true,
  "qc": {
    "approved": true,
    "tileable_score": 0.75
  }
}
```

如果质检失败：

- 重新生成面料
- 或将其修复为可平铺图块
- 或用纯色替换

不得将失败面料推入生产渲染。

## 商业时尚质检

渲染后还应创建 `成品质检报告.json`，标记以下问题：

- 卖点裁片过多
- 所有裁片使用相同纹理或相同密度
- 饰边/细条裁片使用复杂图案
- `裁片填充计划.json` 中缺少 reason
- 引用未批准的图案
- 渲染后的 PNG 没有透明 Alpha
- 色板偏离已批准的商业范围
